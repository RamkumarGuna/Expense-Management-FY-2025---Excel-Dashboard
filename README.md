# Expense-Management-FY-2025---Excel-Dashboard
Expenses Dashboard for FY 2025, designed to help organizations monitor spending, budget utilization, and transaction performance in real time.

This dashboard provides a comprehensive view of financial health, enabling smarter decisions through clear visual insights.

Key highlights from the dashboard:  
💰 Total Expenses: 15,951 K across departments
📊 Budget Allocated: 19,806 K with 3,854 K remaining
📈 Approval Rate: 80.8 % for approved transactions
🧾 Average Expense: 46 K per transaction
🏢 Department Insights: Engineering and HR lead spending with 3,366 K and 3,617 K respectively
💳 Payment Methods: Cheque and Bank Transfer dominate overall payments

Built using Power BI’s data modeling, DAX calculations, and interactive visuals, this dashboard helps finance teams track monthly trends, compare budget vs actuals, and identify opportunities for cost optimization.
